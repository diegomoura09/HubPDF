Tipos de Alertas

Exemplos de Código

Alerta de Sucesso

window.alerts.success('Operação concluída com sucesso!');

Alerta de Erro

window.alerts.error('Erro ao processar sua solicitação.');

Alerta com Link HTML

window.alerts.error(
  'E-mail não encontrado. <a href="/register">Cadastre-se aqui</a>',
  0  // 0 = não fecha automaticamente
);

Uso Avançado

window.alerts.show({
  message: 'Sua mensagem aqui',
  type: 'success',      // success, error, warning, info
  duration: 5000,       // em ms (0 = permanente)
  dismissible: true     // true = pode fechar
});
